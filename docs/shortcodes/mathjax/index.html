    MathJax

    MathJax shortcode let you render math typesetting in markdown document. See MathJax

    Example

    {{< mathjax [display] [class="text-center"]  >}}
x = \begin{cases}
   a &\text{if } b \\
   c &\text{if } d
\end{cases}
{{< /mathjax >}}
    \( x = \begin{cases} a &\text{if } b \\ c &\text{if } d \end{cases} \)
